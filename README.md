# Hotel Booking Analysis & Cancellation Insights (Excel Dashboard)

# Project Overview
This project analyzes hotel booking data to identify booking patterns, customer behavior, and key factors contributing to booking cancellations.  
An interactive dashboard was created using Microsoft Excel to visualize insights.

---

# Objective
To understand:
- Booking trends over time  
- Customer segmentation (Couples, Family, Single)  
- Cancellation behavior  
- Hotel performance (City vs Resort)  

---

# Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning using Excel functions  

---

# Key Metrics
- Total Bookings: 119,390  
- Total Cancellations: 44,224  
- Cancellation Rate: ~37%  

---

# Key Insights

-  Couples contribute the highest number of bookings and cancellations  
-  City Hotels have more bookings compared to Resort Hotels  
-  Cancellation rate is higher in City Hotels  
-  Peak bookings occur during July–August  
-  High cancellations during peak months indicate possible pricing or overbooking issues  
-  Most bookings are for desired room types  

---

# Business Recommendations

- Implement flexible pricing strategies to reduce cancellations  
- Improve booking confirmation policies  
- Focus marketing efforts on the couples segment  
- Optimize inventory planning during peak seasons  

---

# Dataset Features
The dataset includes:
- Hotel type  
- Booking status (Cancelled/Not Cancelled)  
- Arrival date (Year & Month)  
- Guest type  
- Room type  
- Customer details
